Referencias / Registro

Registro de referencia

Formulario dedicado para crear desde cero, copiar referencia o generar una nueva version superior. La clave primaria es codigo referencia + version.

Volver a gestion
Si se actualiza una referencia, la nueva version debe ser superior a la anterior. No se elimina una referencia con registros asociados.

Formulario de referencia
